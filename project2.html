UCF Fashion Society — Club Website

Club Web Development — React, TypeScript, Tailwind CSS, Vite

Repository: UCF-Fashion-Society/Fashion-Society-Web-25-26

Project Overview

Contributed to the official website for the University of Central Florida Fashion Society, a student organization focused on creativity and self-expression through design, styling, modeling, and production. The site serves as the club’s public-facing hub for information about committees, events, and shows.

Key Features

  • Multi-page site with React Router: Home, About, Shows, Events, Committees, and Contact
  • Responsive hero section with custom typography using fluid clamp() sizing
  • Fixed navigation bar with route-aware text color (white on home, dark on inner pages)
  • About page outlining the club’s nine committees and mission
  • Modern styling with Tailwind CSS utility classes

Technical Implementation

  • Framework: React 19 with TypeScript
  • Build tool: Vite for fast development and production builds
  • Routing: React Router DOM for client-side navigation
  • Styling: Tailwind CSS v4 with responsive, mobile-first layouts
  • Collaboration: GitHub organization repository with team-based development

Site Structure

  • src/pages/ — Home, About, Shows, Events, Committees, Contact
  • src/components/ — Shared navigation component with active route styling
  • src/App.tsx — Router configuration and layout shell

Skills Demonstrated

  • React and TypeScript frontend development
  • Responsive UI design with Tailwind CSS
  • Client-side routing and component architecture
  • Collaborative development on an organization-owned GitHub repository
