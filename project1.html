PackThreads — Virtual T-Shirt Collecting Platform

Full-Stack Course Project — JavaScript, React, Node.js, Express, MongoDB

Project Overview

PackThreads is a MERN stack web application where users open virtual graphic T-shirt packs, collect shirts of different rarities, save favorites, and view collection data. Admins can manage the shirt and pack data that powers the app. The project was built collaboratively as a CIS 4004 full-stack development course project.

Key Features

  • User registration and login with JWT authentication
  • Admin and standard user roles with protected admin routes
  • Virtual pack opening with weighted rarity rolls and blank-tag probability mechanics
  • Collection entries, favorites, and per-user leaderboard views
  • Peer-to-peer trading: create offers, accept, decline, and cancel with inventory validation
  • Daily UTC pack slots (two random packs per day) plus bonus pack support
  • Admin dashboard for shirt and pack management

Technical Implementation

  • Frontend: React with Vite, client-side routing, and REST API integration
  • Backend: Node.js and Express with modular controllers, middleware, and route structure
  • Database: MongoDB with Mongoose models for users, packs, shirts, rarities, collections, trades, and opening history
  • Auth: JWT-based authentication with role-based admin middleware
  • Leaderboards: MongoDB aggregation pipelines using $lookup, $addFields, and $sort
  • Game logic: Weighted shirt selection, blank-tag rolls, and single-stitch probability calculations on the server

Architecture

The application follows a standard MERN architecture:

  • client/ — React frontend (pages, components, API layer)
  • server/ — Express API (controllers, models, routes, middleware, seed script)

Skills Demonstrated

  • Full-stack MERN development
  • RESTful API design
  • JWT authentication and role-based access control
  • MongoDB schema design and aggregation queries
  • Probabilistic game mechanics and state management
  • Team collaboration and technical presentation
